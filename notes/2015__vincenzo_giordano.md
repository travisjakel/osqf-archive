## Datasets used

- historical prices May 2010-May 2015

## Code & tools

## Key findings
- Policy events (rule announcement/finalization/effectiveness) show measurable abnormal returns in natural gas contracts  
  > Average Abnormal Returns by Natural Gas contract for Effective events (abnormal returns) 0.4 0.2 0.0 -0.2 NG1 NG2...
- Shorter-dated contracts (NG1-NG3) exhibit higher volatility than longer-dated ones  
  > NG1: Standard Deviation 2.736 vs NG12: 1.446 (volatility summary statistics)
- T-statistics show significant abnormal returns for Effective events in multiple contracts  
  > T−test results for Average Abnormal Returns for Effective Event type (t−statistic) up to +4.0
