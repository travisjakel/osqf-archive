## Key findings
- The Sharpe Ratio formula rewards good returns while penalizing volatility  
  > It rewards good Financial Returns and it penalizes Irratic Behavior of those returns.
- R's quantmod package can retrieve income statement data for thousands of tickers  
  > Income statement figures for thousands of stock tickers can be read via R program using quantmod package
- Income Statement Sharpe Ratio (ISSR) applies Sharpe logic to financial growth metrics  
  > The Income Statement Sharpe Ratio (ISSR) can be computed like Sharpe Ratio of log returns of price history
