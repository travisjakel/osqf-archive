# Testing, Monitoring and Dating Structural Change in FX Regimes

_2010 · Zeileis (University of Vienna, Department of Statistics)_

**Headline:** The talk presents methods for testing, monitoring, and dating structural changes in foreign exchange (FX) regimes using statistical tools like the strucchange and fxregime R packages, with applications to Chinese and Indian exchange rate analysis.

## Key findings
- The Chinese yuan (CNY) was found to still be pegged to the USD in early 2005 despite official announcements of a basket-based regime.  
  > Fitting the model for the first three months (up to 2005-10-31, n = 68) shows that a plain USD peg is still in operation.
- Structural change tests (e.g., Nyblom-Hansen) detected no significant instability in the initial USD peg model (p = 0.697).  
  > which is 1.097 for the Chinese FX regression (p = 0.697).
- Monitoring frameworks using fluctuation processes were applied to track CNY regime changes post-2005.  
  > In 2005: Ajay Shah, Ila Patnaik, and Achim Zeileis establish a webpage and start monitoring the CNY regime.
