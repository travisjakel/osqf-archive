## Key findings
- Machine learning techniques outperformed traditional methods in default prediction  
  > Machine learning techniques have a greater precision than the traditional technique.
- Ensemble methods (Random Forest, AdaBoost) achieved highest accuracy (64.41%)  
  > Ensemble methods, Random Forest and AdaBoost, have done better than the others.
- SVM variants showed comparable performance to ensemble methods  
  > SVM Linear 63,86; SVM Radial 63,87
