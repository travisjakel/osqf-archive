# Analysis of Predictive Power of Click Data on Fund Flow Using R

_2018 · Yu Li (Discover Financial Services, Senior Associate, Consumer Banking Digital Collection)_

**Headline:** Analysis of how Morningstar click/visit data can predict mutual fund flows using R

## Topics
- predictive power of click data
- fund flow prediction
- R programming applications
- mutual fund analysis
- financial data modeling

## Methods
- simple linear regression
- principal component analysis (PCA)

## Datasets used
- Nine funds' click-rate/visit-rate data (10/17/2016-7/14/2017)
- Morningstar product data with benchmarks and ratings

## Code & tools
- R

## Key findings
- Adjusted R-squared values for 5 funds were near zero, leading to their exclusion  
  > Table 1 shows five funds' Adjusted R-Squared of all models are close to zero. These funds were dropped.
- Only four funds (OAKIX, ANWPX, CBLCX, VILLX) showed moderate predictive power improvements  
  > The other four funds were selected for further analysis, which are: OAKIX, ANWPX, CBLCX, and VILLX
- Predictive power remained low despite using click/visit data derivatives  
  > Overall, the click-rate, visit-rate and their derivative variables could help to increase the predictive power on fund flow, while the predictive power is still not high enough.

## Applications
- enhancing mutual fund flow prediction models
