## Key findings
- Sample covariance matrix should not be used for portfolio optimization  
  > "The central message of this paper is that nobody should be using the sample covariance matrix for the purpose of portfolio optimization."
- Random portfolios methods improve optimization robustness  
  > Comparison of Random Portfolio Methods (Interactive!)
- Factor models provide better moment estimates than raw data  
  > Cleaned returns data and factor model moment estimates
