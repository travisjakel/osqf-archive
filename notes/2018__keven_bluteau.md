# Abnormal Tone and Abnormal ReturnsAn Event Study Analysis

_2018 · Keven Bluteau (University of Neuchâtel & Vrije Universiteit Brussel)_

**Headline:** Analyzing abnormal tone using event study methods in the context of earnings announcements.

## Methods
- event study econometric tools
- regression analysis of tone on explanatory factors
- weighted market/industry tone indices
- Word Power methodology (Jegadeesh & Wu, 2013)

## Datasets used
- S&P500 firms (2000Q1-2016Q4)
- 19,319 quarterly earnings announcements
- 3 million text publications

## Code & tools
- Word Power methodology

## Key findings
- Cumulative abnormal tone (CAT) shows upward trends even with negative earnings surprises.  
  > An upward CAT even for negative earnings surprise at the event date
- CAT trends differ from cumulative abnormal returns (CAR) patterns.  
  > Slow downtrend for CAT, not consistent with CAR

- financial event analysis
