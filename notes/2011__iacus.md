## Key findings
- Markov Operator distance (MO) outperforms traditional distance measures for clustering financial time series.
- Quasi-MLE (QMLE) with the 'yuima' package provides accurate parameter estimation for diffusion processes, even with small sample sizes.
- AIC-based model selection effectively balances model complexity and fit for diffusion processes.
- Two-stage least squares (2SLS) improves parameter estimation in nonlinear interest rate models.

## Applications
- Clustering financial assets using Markov Operator distance
- Parameter estimation for Ornstein-Uhlenbeck processes (QMLE)
- Interest rate modeling with Aït-Sahalia's polynomial specification (2SLS)
- Model selection for diffusion processes using AIC
