## Key findings
- SWIFT data provides standardized cross-border transaction records suitable for network analysis  
  > SWIFT is used in cross-border transactions, HSVP (RTGS in Croatia), TARGET2... international standardized message format
- Financial network analysis reveals structural changes in interbank systems post-EU accession  
  > network metrics used for statistical analysis... closeness centrality of interbank wire transfers with EEA countries
- The infrastructure requires minimal resources and is usable by institutions with limited IT budgets  
  > The infrastructure is simple to implement, requires low maintenance... small and limited IT budgets
