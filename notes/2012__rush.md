## Key findings
- Restricted GLS model showed improved log likelihood compared to full model.  
  > Restricted Model Log Likelihood 917.5741 vs Full Model 881.9172
- Return spreads significantly correlated with VIX and treasury rate spreads.  
  > 1yr Treasury coefficient 21.64806 (p=0.0000), 10yr-2yr coefficient 40.71688 (p=0.0000)
- Return spread differential explained 11.63% annualized returns.  
  > Expected Annual Return 11.63% with 80.22% probability of positive annual return
