## Key findings
- OneTick enables R-based analysis of cross-day volume profiles

- R can access OTCloud data via REST API  
  > URL WebAPI from R: Design & Save Query, Build URL, Call from R
- OneTick supports execution, news, and derived data analysis  
  > Supports Executions, News, Derived and Unstructured data
