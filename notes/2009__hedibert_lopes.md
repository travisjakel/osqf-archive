## Key findings
- Particle Learning avoids particle degeneracy by using conditional sufficient statistics for parameter estimation.
- PL achieves lower mean absolute error (MAE) than SISR in sequential parameter estimation tasks.
- Reverse Kalman filter logic in PL improves efficiency by using future data in both resampling and propagation steps.
